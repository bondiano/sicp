## Sicp Racket Boilerplate

### Requirements

* Racket
* Rackunit (`raco pkg install rackunit`)
* Make

### Using

1. Copy content of this repo (not clone!) to your repository
1. Write code (look `sicp/chapter1/02.rkt`)
1. Run tests `make test`
